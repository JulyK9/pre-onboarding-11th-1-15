> AngularJS 커밋 컨벤션을 기반으로 따릅니다.

## \***\*Commit Message Convention\*\***

### 커밋 메세지의 기본 형식

```jsx
<type>(<scope>): <short summary>
<BLANK LINE>
<body>
<BLANK LINE>
<footer>
```

### 타입**에 들어갈 수 있는 항목들**

<aside>
📌 docs : 문서 관련

style : 스타일 변경 (포매팅 수정, 들여쓰기 추가, …)

refactor : 코드 리팩토링

test : 테스트 관련 코드

build : 빌드 관련 파일 수정

ci : CI 설정 파일 수정

perf : 성능 개선

chore : 그 외 자잘한 수정

</aside>
